# API

The API section is organized by modality and function, similar to Scanpy-style docs:

- **RNA**: preprocessing, cell typing, sample embedding, resolution selection
- **ATAC**: ATAC preprocessing plus shared cell typing/embedding/resolution functions
- **Multi-omics**: GLUE integration, integrated preprocessing, multi-omics cell typing, embedding, and optimization
- **Downstream**: distance, trajectory, clustering, tests, and visualization

Each function page includes:

- function signature
- default values
- what each parameter controls
- outputs and common usage notes

If a function is shared across categories, it is documented once and linked from other categories.
